Используемые технологии :
Java
Maven
Spring boot
PostgresSQL
Angular

 Описание:

 Приложение получает список актуальных валют с сайта ЦБРФ http://www.cbr.ru/scripts/XML_daily.asp и записывает в БД.
 пользователь пройдя авторизацию попадает на главный экран, где может выбрать из какой валюты и в какую будет конвертация,
 указывает количество переводимых средств и нажимает кнопку "Конвертировать". После чего происходит запрос в БД на получение актуального курса
 на текущую дату.акже в конвертере ведется запись операций в таблицу.
